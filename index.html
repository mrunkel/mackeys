Convert Apple Mac keys to their single-Unicode-character equivalents

About

This tool converts easy-to-type textual descriptions of keyboard keys into the Unicode characters representing those keys. For instance, entering the following:

Press Cmd + Shift + 4 to take a screenshot.

…would return…

Press ⌘ + ⇧ + 4 to take a screenshot.

This can be useful for tweets, where every character counts. Personally, I use it because it looks better.

An iPhone-optimized version of this page is available. To view it, just visit this page using an iPhone.
